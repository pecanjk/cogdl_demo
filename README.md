# Vue3 模型库管理系统

一个基于Vue3的现代化模型库管理系统，参考HuggingFace的设计风格，提供模型展示、上传、编辑和从训练任务自动创建模型的功能。

## ✨ 特性

### 🎯 核心功能
- **模型库展示** - 类似HuggingFace的卡片式展示，支持搜索、过滤和排序
- **在线编辑** - 支持模型信息编辑，包含Markdown编辑器和实时预览
- **模型上传** - 完整的模型上传流程，支持多文件上传和进度显示
- **任务集成** - 从训练任务自动识别和创建模型，智能生成模型信息

### 🎨 设计特色
- **HuggingFace风格** - 现代化的UI设计，优雅的交互体验
- **响应式布局** - 完美适配桌面和移动设备
- **智能识别** - 基于任务信息自动推断模型类型、标签等
- **实时预览** - Markdown编辑器支持实时预览

### 🔧 技术特点
- **Vue 3** + **Composition API** - 现代化的前端框架
- **Element Plus** - 企业级UI组件库
- **Pinia** - 轻量级状态管理
- **Vite** - 快速的构建工具
- **SCSS** - 强大的CSS预处理器

## 🎯 功能详解

### 模型库展示
- **卡片式布局** - 每个模型以卡片形式展示，包含关键信息
- **搜索功能** - 支持按名称、描述、作者搜索
- **标签过滤** - 多选标签过滤功能
- **排序选项** - 支持按更新时间、下载量、点赞数排序
- **响应式网格** - 自适应不同屏幕尺寸

### 模型详情页
- **完整信息展示** - 模型的详细信息和统计数据
- **文件列表** - 显示模型包含的所有文件
- **README渲染** - Markdown格式的说明文档
- **操作按钮** - 下载、点赞、编辑等操作

### 模型上传
- **分步骤表单** - 清晰的信息填写流程
- **文件拖拽上传** - 支持拖拽和点击上传
- **进度显示** - 实时显示上传进度
- **Markdown编辑器** - 支持编辑和预览README

### 任务集成
- **任务列表** - 展示所有训练任务
- **智能识别** - 自动识别可创建模型的任务
- **一键创建** - 从任务快速创建模型
- **智能建议** - 基于任务信息生成模型信息建议

## 🔄 数据流

### 模型管理流程
1. **获取模型列表** - 从API获取模型数据
2. **搜索和过滤** - 客户端实时过滤
3. **查看详情** - 获取单个模型完整信息
4. **编辑更新** - 提交更新到服务器
5. **上传新模型** - 完整的上传流程

### 任务集成流程
1. **获取任务列表** - 从API获取训练任务
2. **智能分析** - 分析任务信息生成建议
3. **创建模型** - 基于任务数据创建模型
4. **自动关联** - 关联任务和模型数据

## 🔧 API接口

### 模型相关
```javascript
// 获取模型列表
GET /api/models

// 获取模型详情
GET /api/models/:id

// 创建模型
POST /api/models

// 更新模型
PUT /api/models/:id

// 删除模型
DELETE /api/models/:id

// 点赞模型
POST /api/models/:id/like

// 下载模型
GET /api/models/:id/download

// 上传模型文件
POST /api/models/upload
```

### 任务相关
```javascript
// 获取任务列表
GET /api/tasks

// 获取任务详情
GET /api/tasks/:id

// 从任务创建模型
POST /api/tasks/:id/create-model
```
